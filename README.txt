# 🧾 Visitor Management System (PHP + MySQL)

A web-based visitor management system built using PHP and MySQL. It allows organizations to log, manage, and monitor visitor entries efficiently through a secure admin dashboard. Designed with responsive UI and basic authentication.

---

## 🔍 Features

- 🟢 Visitor check-in and check-out system
- 🔐 Admin login system for secure access
- 📋 Add, update, and delete visitor entries
- 🖨️ Option to print visitor passes
- 📊 Display visitor history
- 💡 Simple and user-friendly interface
- 📱 Fully responsive layout using Bootstrap

## 🚀 Getting Started

### Prerequisites

- [XAMPP](https://www.apachefriends.org/index.html) or any PHP + MySQL local server

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/sujalverma7557/visitor-management-php.git

Move the project folder

Copy or move it into your htdocs/ directory of XAMPP.

Import the database

Open phpMyAdmin

Create a new database, e.g., visitor_db

Import the provided db.sql file (if available)

Configure database (if needed)

Open dbcon.php and check database name, username (root), and password ("" by default in XAMPP)

Run the application

http://localhost/visitor-management-php
